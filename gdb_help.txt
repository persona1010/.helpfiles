#run - Старт/рестарт выполнения программы.
#break, b [<line num, func name>] - устанавливает брейкпоинт.
#next, n - Следущая инструкция.
#list - Выводит часть кода вокруг текущей позиции.
#print, p <var> - Выводит  значения переменных.
#quit
#up - Перемещает вверх по стаку вызовов.
#down - Перемещает вниз по стаку вызовов.
#display <val> - Следить за петеменной.
#undisplay <id> - Перестать отслеживать дисплей по номеру id.
#backtrace - Выводит стак вызовов.
#step - Входит в функцию (step into).
#continue - Продолжить до слудующего брейкпоинта.
#finish - Остановка после возврата из функции?
#watch <var> - Следит за переменной и останавливается при изменении 
    отслеживаемого элемента.
#info breakpoints - Выводит список брейкпоинтов.
#delete [<id>] - Удаляет все брейкпоинты, или по id, который отображает 
    комманда info.
#what <var> - Отображает тип переменной.
#target record-full - Дает возможность возвращатся к предъидущим линиям кода.
#reverse-next - Некст в обратром направлении.
#reverse-step
#reverse-continue
#set var <var> - Устанавливает новое значение для переменной.
